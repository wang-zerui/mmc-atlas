ConfLayers (Amer et al. 2026)

M196arXiv 2026verified source ↗
Training-free self-speculative decoding that forms the drafter by confidence-based intermediate-layer skipping with an adaptive threshold, with the full target still verifying, so the mechanism is intended distribution-preserving.

Background

Self-speculative decoding pairs fast approximate decoding (a compact draft built from the model itself) with selective re-checking by the full target model.

Problem

Build the layer-skipping drafter for self-speculative decoding without training a learned skip policy.

Motivation / Analysis

Heuristic-based layer selection can outperform learned skipping policies while avoiding the overhead and complexity of training a policy.

Key Insight

Confidence scores across layers, combined with an adaptive threshold, can select which intermediate layers to skip on the fly, yielding an effective drafter with no trained policy.

Challenges

Choosing the adaptive confidence threshold; iteratively searching layer configurations without training; staying robust across tasks and datasets.

Method

ConfLayers is a plug-and-play approach that builds the draft model via confidence-based intermediate-layer skipping: it computes per-layer confidence scores, selects layers to skip via an adaptive threshold, evaluates the configuration, and iteratively updates the best selection until improvement stalls or a max iteration count; the full target model verifies drafted tokens.

Experimental Setup

Evaluated across different models and datasets against vanilla LLM generation (per abstract).

Results

Up to 1.4x speedup over vanilla LLM generation (verified in claim_checked). The full target verifies, so the mechanism is intended distribution-preserving; however, the authors frame it as a speed-quality trade-off and do not use the word 'lossless', so it is best described as near-lossless.

Limitations

Modest speedup (up to ~1.4x); authors frame results as a speed-quality trade-off rather than strictly lossless; arXiv preprint.

Relation to the Field

A training-free, adaptive version of Self-Speculative Decoding (Zhang et al.) layer-skip drafting, avoiding the learned skip policies used by prior layer-skipping methods.
