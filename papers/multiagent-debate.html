Multiagent Debate (Du et al. 2023)

M002ICML 2024verified source ↗
Multiple LLM instances independently propose answers and then debate over several rounds until they converge on a shared final answer, improving factuality and reasoning without any training.

Background

LLMs are strong single-pass reasoners but still hallucinate and make reasoning errors, and prior test-time methods (CoT, self-consistency) rely on a single model's own samples. This work borrows Minsky's 'society of minds' idea, using several model instances as interacting agents.

Problem

How can black-box LLM performance on mathematical, strategic, and factual tasks be improved at inference time without fine-tuning or external tools?

Motivation / Analysis

If several model instances reason independently and then critique each other, errors that are idiosyncratic to one chain should be corrected during debate, yielding more robust consensus answers.

Key Insight

Exposing each agent to the other agents' reasoning across multiple rounds lets them revise and converge, and the procedure works cross-model (e.g. ChatGPT + Bard) and scales with the number of agents and rounds.

Challenges

Managing debate context length across rounds, ensuring agents actually revise rather than blindly agree, and keeping the same prompt/procedure general across diverse tasks.

Method

Each of N model instances generates an initial response; in each subsequent round every agent is shown the other agents' answers and reasoning and produces an updated response; after a fixed number of rounds the converged answer is taken. The identical procedure and prompts are used across all tasks and applies directly to existing black-box models.

Experimental Setup

Evaluated on mathematical reasoning, strategic reasoning, and factual-validity/hallucination tasks; ablates number of agents and number of rounds, and demonstrates cross-model debate.

Results

The abstract reports that debate 'significantly enhances mathematical and strategic reasoning' and improves factual accuracy while reducing false/hallucinated answers; performance increases with more agents and more rounds and holds across models (ChatGPT+Bard). Specific per-benchmark numbers are given in the full paper, not the abstract.

Limitations

Debate multiplies inference cost with the number of agents and rounds; the abstract does not quantify per-task gains, and later work (Huang et al. 2023) questions whether gains survive matched sample budgets against self-consistency.

Relation to the Field

Foundational debate method for this family; directly reproduced and critiqued by 'LLMs Cannot Self-Correct Reasoning Yet' (M059) and by 2026 negative-result papers (M158, M159), and extended for efficiency by Sparse Communication Topology (M063).
