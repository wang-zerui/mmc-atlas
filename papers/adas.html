ADAS / Meta Agent Search (Hu et al. 2024)

M100ICLR 2025verified source ↗
ADAS (Automated Design of Agentic Systems) has a meta agent write agents as code, evaluate them, and store them in a growing archive, iteratively inventing novel building blocks and workflows in a Turing-complete search space.

Background

In ML history, hand-designed components are eventually replaced by learned ones; agentic systems were still assembled by hand from human-designed modules like Chain-of-Thought and Self-Reflection.

Problem

Designing powerful agentic systems (prompts, tool use, workflows and their combinations) required substantial human expertise and trial-and-error, limiting the space of designs explored.

Motivation / Analysis

If agents are expressed in code and a meta agent can program them, the search space becomes Turing-complete, theoretically covering any possible agentic system and enabling automatic discovery of new designs.

Key Insight

Defining agents in code lets a meta agent invent, run, and refine entirely new agents, and an ever-growing archive of prior discoveries lets it build progressively better designs on top of earlier ones.

Challenges

Searching an unbounded code space efficiently, evaluating candidate agents reliably, and ensuring discovered agents generalize rather than overfit to the search benchmark.

Method

Meta Agent Search: a meta agent (GPT-4-class) iteratively programs new agents in code, evaluates them on target tasks (evaluation with GPT-3.5), and adds successful designs to an archive that seeds subsequent iterations (on the order of 25 iterations).

Experimental Setup

Ran across coding, science, and math domains, and tested transfer of discovered agents across held-out domains and different underlying models.

Results

Discovered agents greatly outperform state-of-the-art hand-designed agents and maintain superior performance when transferred across domains and models; the abstract reports these gains qualitatively without a single headline metric.

Limitations

The abstract does not report specific numerical margins; the search is compute-intensive and the quality of discovered agents depends on the meta agent's coding ability and the reliability of the evaluator.

Relation to the Field

A code-space search counterpart to graph-search methods like GPTSwarm and MCTS-based AFlow, which explicitly reports large gains over ADAS on MATH-lv5/MBPP; a precursor to holistic MAS optimizers (OMAC, MAS-Orchestra).
