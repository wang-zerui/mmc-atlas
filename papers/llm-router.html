LLM Router: Rethinking Routing with Prefill Activations (Varshney et al. 2026)

M123arXiv 2026verified source ↗
Routes queries using a model's internal prefill activations instead of surface query embeddings, using Encoder-Target Decoupling so open-weight encoders can predict the correctness of closed-source models.

Background

Routers usually decide from query text embeddings, discarding the rich internal signals a model computes while reading the prompt.

Problem

Query embeddings are a weak predictor of whether a given model will answer correctly, and you cannot access closed-source models' internals.

Motivation / Analysis

A model's prefill activations encode how it 'understands' a query and should better predict its own correctness, but closed models expose no activations.

Key Insight

Prefill activations from an open-weight proxy encoder are a strong routing signal, and decoupling the encoder from the target model lets those activations predict closed-source model correctness.

Challenges

Selecting which layers carry the predictive signal (addressed via Fisher-Separability layer selection) and sharing representation across targets (via a SharedTrunkNet).

Method

Encoder-Target Decoupling uses an open-weight encoder's prefill activations, selects informative layers by Fisher separability, and feeds them to a SharedTrunkNet that predicts each target model's correctness for routing.

Experimental Setup

Routing over closed and open models using prefill-activation features; measures the gap closed to the oracle and cost savings versus always using the strongest model.

Results

Closes 45.58% of the strongest-model-to-oracle gap at 74.31% cost savings (verified in claim_checked); submitted 21 Mar 2026 (v2 31 Mar), no venue claimed.

Limitations

Requires running an open-weight encoder's prefill for every query, adding overhead; preprint without a venue.

Relation to the Field

Introduces an activation-based routing signal, contrasting with embedding-based routers (RouteLLM, RouterDC) and confidence-based cascades; a distinctive 'internal-signal' approach in this family.
