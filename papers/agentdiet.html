AgentDiet (Xiao et al. 2025)

M092FSE 2026pending audit source ↗
An inference-time trajectory-reduction method that automatically prunes redundant, useless, and expired information from a software agent's growing context to cut input-token cost without hurting performance.

Background

Multi-turn LLM agents for software engineering accumulate long trajectories, and the growing context drives up input-token cost at every step.

Problem

Reduce the input-token cost of long-horizon SWE agents while maintaining the same task performance.

Motivation / Analysis

Agent trajectories often contain wasteful information (redundant, useless, or expired) that can be removed without affecting the agent's decisions.

Key Insight

Automatically identifying and pruning wasteful trajectory content at inference time can substantially cut token cost while keeping agent behavior unchanged.

Challenges

Deciding which trajectory content is safe to remove without degrading performance; doing so cheaply during execution.

Method

AgentDiet is a simple, inference-time trajectory reduction approach that detects and removes useless, redundant, and expired information from the agent's context as the trajectory grows during execution.

Experimental Setup

Evaluated with two LLMs on two agent benchmarks (per claim_checked, Trae Agent on SWE-bench Verified); the abstract does not name the benchmarks explicitly.

Results

Reduces input tokens by 39.9%-59.7% and total computational cost by 21.1%-35.9% while maintaining the same agent performance (verified in claim_checked; magnitudes not independently re-run). As context pruning it is lossy by construction, but reported to preserve task performance.

Limitations

Lossy context removal; performance-preservation demonstrated on two LLMs/benchmarks and not independently audited; risk of pruning useful context if the heuristic misfires.

Relation to the Field

An agent-level runtime strong-weak delegation of context reduction, related to LLMLingua/LLMLingua-2 prompt compression and ACON's agent context optimization.
