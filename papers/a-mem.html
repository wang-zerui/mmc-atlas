A-MEM: Agentic Memory for LLM Agents (Xu et al. 2025)

M099NeurIPS 2025verified source ↗
A-MEM is a Zettelkasten-inspired agentic memory in which each memory is an atomic note with LLM-generated keywords, tags, and contextual description, dynamically linked to related notes and allowed to evolve as new memories arrive - moving beyond static RAG retrieval.

Background

LLM agents rely on memory systems to persist and recall information across interactions, but most offer only basic storage and retrieval.

Problem

Existing memory systems lack sophisticated organization: they store and fetch entries but do not build or maintain a structured, interconnected knowledge network.

Motivation / Analysis

The Zettelkasten note-taking method shows how atomic, interlinked notes can form an evolving knowledge network, suggesting a richer organizing principle for agent memory.

Key Insight

Combining Zettelkasten structure with agent-driven decisions lets memory organize itself: new notes generate their own attributes, link to semantically related past notes, and can trigger updates to the attributes of existing memories.

Challenges

Deciding which past memories to link, generating useful structured attributes automatically, and propagating updates through the network without corrupting prior knowledge.

Method

For each new memory, the system generates a structured note (contextual description, keywords, tags), links it to existing memories where meaningful similarities exist, and lets the addition trigger memory evolution by updating the representations and attributes of related historical notes.

Experimental Setup

Empirically evaluated across six foundation models and compared against existing state-of-the-art memory baselines.

Results

The paper reports superior improvement over existing SOTA memory baselines across the six foundation models; specific per-benchmark metric values are not stated in the abstract. Verified: dynamic indexing/linking plus memory evolution, six-model evaluation, NeurIPS 2025.

Limitations

The abstract does not report the exact magnitude of gains or the cost/latency of LLM-driven note generation and evolution; benefits may vary with the base model's structuring ability.

Relation to the Field

A structure- and evolution-centric alternative to context-paging memory like MemGPT; both are single-agent memory systems, whereas the Mesh Memory Protocol addresses cross-agent memory sharing.
