TwinRouterBench (Yang et al. 2026)

M127arXiv 2026verified source ↗
A step-level routing benchmark for long-horizon agentic apps with a static deterministic track (970 router-visible prefixes) and a live dynamic track running routers on SWE-bench Verified to measure real task resolution and API spend.

Background

Agentic apps (coding, research agents) make many model calls over a long trajectory, so routing decisions happen at the step level, not per standalone query.

Problem

Existing routing benchmarks evaluate single queries and cannot measure how step-level routing affects end-to-end agentic task success and cost.

Motivation / Analysis

To fairly evaluate agentic routers you need both reproducible static comparison and a live setting that captures compounding step decisions on real tasks.

Key Insight

Pairing a deterministic replay track (fixed prefixes) with a live execution track lets routers be compared reproducibly and also measured on real task resolution and spend.

Challenges

Building 970 router-visible step prefixes from SWE-bench/BFCL and orchestrating a live track that runs routers inside real agents on SWE-bench Verified.

Method

Two tracks: a static deterministic track with 970 router-visible prefixes drawn from SWE-bench and BFCL, and a live dynamic track that runs routers end-to-end on SWE-bench Verified (520 instances), measuring task resolution rate and API dollar spend.

Experimental Setup

Static track: 970 prefixes; live track: SWE-bench Verified with 520 instances, measuring real resolution and spend for candidate routers.

Results

Confirms the two-track agentic step-level design, 970 prefixes / 520 instances, and live SWE-bench Verified spend measurement (verified in claim_checked); v1 14 May 2026 (v2 22 May), no venue claimed.

Limitations

Live-track results depend on the specific agents and model pool used; preprint without a venue.

Relation to the Field

Extends routing benchmarking (RouterBench M028, LLMRouterBench M122) into the agentic step-level regime, aligned with agentic routers like Switchcraft (M124).
