BASTION (Oh et al. 2026)

M192arXiv 2026verified source ↗
Budget-aware speculative decoding with tree-structured block-diffusion drafting that adaptively grows per-query draft trees using an acceptance surrogate and a hardware roofline latency estimator; training-free and lossless.

Background

Block-diffusion drafters make parallel predictions from position-wise marginals rather than fully conditioned sequences, so greedily choosing one path can miss the target model's preferred trajectory.

Problem

Build higher-quality draft trees from block-diffusion drafters while respecting hardware/latency budgets, without training.

Motivation / Analysis

Static tree topologies cannot adapt to per-query difficulty or hardware limits; the tree should grow only while added verification cost is worth the expected gain.

Key Insight

Query-dependent trees that balance draft quality against a hardware-aware latency budget, guided by an acceptance surrogate, capture the target's preferred paths better than static marginal drafting.

Challenges

Estimating expected accepted length online; calibrating a hardware roofline latency model; deciding when to stop expanding the tree.

Method

BASTION builds query-dependent draft trees combining (1) an acceptance surrogate estimating expected accepted length via path confidence, (2) an online latency estimator calibrating a hardware-aware roofline model, and (3) adaptive best-first expansion that grows the tree until added verification cost outweighs gains; it is training-free.

Experimental Setup

Evaluated against standard autoregressive decoding and leading block-diffusion speculative baselines (per abstract).

Results

Up to 6.61x speedup over autoregressive decoding and +39% over leading block-diffusion baselines (verified in claim_checked). The framework is training-free and preserves the target model's distribution, so it is distribution-preserving (lossless), with no per-setting tuning.

Limitations

Targets block-diffusion drafters specifically; relies on accurate online latency/acceptance estimation; arXiv preprint not yet peer-reviewed.

Relation to the Field

Extends tree-based speculative decoding (SpecInfer) to block-diffusion drafting with budget-awareness; TreeFlash addresses the same marginal-distribution drift of parallel/tree drafting via AR-approximation.
