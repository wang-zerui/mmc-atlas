Self-Refine (Madaan et al. 2023)

M014NeurIPS 2023verified source ↗
A single LLM plays generator, feedback-provider, and refiner, iteratively critiquing and improving its own output at test time with no extra training.

Background

Like humans, LLMs rarely produce their best output on the first attempt, yet most decoding pipelines are single-pass. Prior refinement approaches required supervised data or reward models.

Problem

Can an LLM improve its own outputs iteratively using only self-generated feedback, without any supervised data, additional training, or reinforcement learning?

Motivation / Analysis

If a model can articulate what is wrong with its own draft, that feedback can be fed back as a refinement instruction, creating a cheap test-time improvement loop reusing one model.

Key Insight

The same frozen LLM is capable enough to critique its own output and act on that critique, so generation, feedback, and refinement can all be the same model in a loop.

Challenges

Getting actionable rather than vacuous self-feedback, deciding when to stop iterating, and ensuring refinement does not degrade already-good outputs.

Method

One LLM generates an initial output, then produces natural-language feedback on that output, then refines the output conditioned on the feedback; the feedback-refine cycle repeats for several iterations at inference time, with no fine-tuning.

Experimental Setup

Evaluated on 7 diverse tasks spanning dialogue response generation to mathematical reasoning, using GPT-3.5, ChatGPT, and GPT-4, with both human preference and automatic metrics.

Results

Self-Refine outputs are preferred over one-step generation across all 7 tasks, with roughly 20% absolute average improvement in task performance, and it improves even GPT-4 at test time.

Limitations

Relies on the base model being able to generate useful self-feedback; gains vary by task, iteration count and stopping criterion are heuristic, and it adds inference cost per refinement round.

Relation to the Field

Canonical single-model self-critique baseline for this family; contrasts with multi-agent debate (M002) and is the 'isolated self-correction' style shown to sometimes beat unguided debate (M158), while intrinsic self-correction of reasoning is itself questioned by M059.
