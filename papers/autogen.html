AutoGen (Wu et al. 2023)

M010arXiv 2023 (later COLM 2024 workshop / widely cited)verified source ↗
AutoGen is an open-source framework of customizable, conversable agents that solve tasks by messaging one another, mixing LLMs, tools, and human input.

Background

LLM-based applications increasingly require orchestrating multiple capabilities (reasoning, tool use, human oversight) rather than a single prompt-response loop.

Problem

Building multi-agent LLM applications was ad hoc; there was no general infrastructure to define agents and program the conversations between them.

Motivation / Analysis

A unified abstraction where agents converse would let developers compose complex workflows from reusable building blocks across many domains.

Key Insight

Treating every participant (LLM, tool executor, human) as a 'conversable agent' with configurable roles turns application logic into programmable multi-agent conversation patterns.

Challenges

Supporting flexible interaction modes, mixing natural language and code to steer conversations, and keeping the framework generic across task complexity.

Method

Provides conversable agents (e.g. AssistantAgent, UserProxyAgent) whose behaviors are customizable; developers program conversation patterns via both natural language and code, and agents can invoke tools or defer to humans.

Experimental Setup

Empirical studies across six domains: mathematics, coding, question answering, operations research, online decision-making, and entertainment.

Results

The abstract reports effectiveness across all six domains but gives no single headline benchmark number; the verified claim is the ConversableAgent/AssistantAgent/UserProxyAgent design and multi-domain applicability.

Limitations

The paper demonstrates applicability rather than reporting head-to-head accuracy gains; performance depends heavily on how developers design the conversation patterns and prompts.

Relation to the Field

A foundational orchestration framework; contrasts with role-fixed pipelines like MetaGPT/ChatDev and later dynamic orchestrators (Magentic-One is built on AutoGen).
