AdaptOrch: Task-Adaptive Multi-Agent Orchestration in the Era of LLM Performance Convergence (Yu 2026)

M152arXiv 2026 (preprint)pending audit source ↗
AdaptOrch argues that as provider LLMs converge on benchmarks the orchestration topology, not model choice, dominates, and dynamically routes each task DAG to a parallel/sequential/hierarchical/hybrid topology.

Background

LLMs from different providers are converging to similar benchmark performance, reducing the payoff of picking a single best model per task.

Problem

When model capabilities converge, per-task model selection yields diminishing returns, yet most systems still fix a single orchestration topology.

Motivation / Analysis

If topology drives system-level performance more than model choice, orchestration design should be treated as a first-class optimization target.

Key Insight

A 'Performance Convergence Scaling Law' defines when orchestration selection matters more than model selection, so routing each task's decomposition DAG to the right topology beats any fixed topology.

Challenges

Mapping task dependency graphs to optimal topologies efficiently and synthesizing consistent outputs from parallel agents with termination guarantees.

Method

Three components: a Performance Convergence Scaling Law; a Topology Routing Algorithm mapping decomposition DAGs to parallel/sequential/hierarchical/hybrid patterns in O(|V|+|E|) time; and an Adaptive Synthesis Protocol with termination guarantees and heuristic consistency scoring.

Experimental Setup

Validated on coding (SWE-bench), reasoning (GPQA), and retrieval-augmented generation tasks using identical underlying models.

Results

Reports a 12-23% improvement over static single-topology baselines even with identical underlying models (title and method confirmed vs abstract; arXiv preprint only, numbers not audited).

Limitations

arXiv preprint with no venue and single-author; the 12-23% range is not independently audited and relies on heuristic consistency scoring.

Relation to the Field

Complements MacNet's topology findings and AOrchestra's dynamic composition, reframing model-convergence as motivation for topology-first orchestration.
