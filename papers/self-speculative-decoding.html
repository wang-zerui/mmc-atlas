Self-Speculative Decoding / Draft & Verify (Zhang et al. 2023)

M083ACL 2024verified source ↗
The model drafts by selectively skipping its own intermediate layers, then verifies those tokens with the full model in one forward pass, making it draft-model-free and lossless.

Background

Speculative decoding accelerates LLMs but typically requires a separate, well-matched draft model that adds memory and training/engineering overhead.

Problem

Achieve speculative-style acceleration without any additional model, extra training, or extra memory footprint.

Motivation / Analysis

A subset of a model's own layers can produce fast, slightly-degraded draft tokens, so the model can serve as its own drafter.

Key Insight

Skipping intermediate layers during drafting gives a cheap approximation of the model, and a single full-model forward pass can verify the drafted tokens so the final output is identical to the unaltered LLM.

Challenges

Choosing which layers to skip to balance draft speed and acceptance; ensuring verification keeps output exactly identical to standard decoding.

Method

A two-stage self-speculative scheme: the drafting stage generates tokens by selectively skipping certain intermediate layers; the verification stage runs the full model once to accept/reject drafted tokens, guaranteeing identical output.

Experimental Setup

Evaluated on LLaMA-2 and its variants across generation benchmarks.

Results

Up to 1.99x speedup on LLaMA-2 (verified in claim_checked). Verification makes output identical to the unaltered LLM, so the method is distribution-preserving (lossless), with no additional training or memory.

Limitations

Speedup (up to ~2x) is more modest than draft-model or tree-based methods; the optimal layer-skip configuration must be selected per model.

Relation to the Field

A draft-model-free, plug-and-play alternative to speculative decoding; the layer-skip drafting idea is later made training-free/adaptive by ConfLayers.
