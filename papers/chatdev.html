ChatDev (Qian et al. 2023/ACL 2024)

M050ACL 2024verified source ↗
ChatDev is a chat-powered software-development framework where role-specialized LLM agents (CEO/CTO/programmer/tester) collaborate along a 'chat chain' with communicative dehallucination to build software end-to-end.

Background

Software development needs collaboration among people with different skills, but prior deep-learning approaches targeted individual waterfall phases with distinct designs.

Problem

Phase-specific designs cause technical inconsistencies across design, coding, and testing, fragmenting the development process.

Motivation / Analysis

Using natural language as a unifying medium across all phases could keep the process coherent and let a single framework span the whole lifecycle.

Key Insight

A 'chat chain' structures what agents communicate (dividing each phase into ordered subtask dialogues), while 'communicative dehallucination' structures how they communicate to catch and correct errors; natural language aids design and programming-language communication aids debugging.

Challenges

Preventing coding hallucinations and inconsistencies propagating across phases while keeping the pipeline autonomous.

Method

LLM-driven agents in specialized roles (CEO, CTO, programmer, reviewer, tester) proceed through design, coding, and testing via multi-turn role-played dialogues organized as a chat chain, with a dehallucination protocol for reliable exchanges.

Experimental Setup

Evaluated on autonomous software-generation tasks; case studies of full application generation with cost/time reporting.

Results

Verified claim: chat-chain + communicative-dehallucination design; ChatDev can complete a full software application in under 7 minutes for under $1 (headline efficiency figures from the paper/ACL 2024).

Limitations

Generated software is small-scale and may contain bugs; the role script and waterfall structure are fixed rather than adaptively composed.

Relation to the Field

A flagship role-specialized software-team system alongside MetaGPT; its authors later scaled agent collaboration into graph topologies (MacNet).
