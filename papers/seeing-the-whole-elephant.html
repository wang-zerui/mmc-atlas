Seeing the Whole Elephant / TraceElephant (Chen et al. 2026)

M227ACL 2026verified source ↗
A benchmark for failure attribution in LLM multi-agent systems that provides complete execution traces and reproducible environments to identify which agent and which step caused a failure.

Background

Failure attribution in LLM multi-agent systems is hard because of natural-language reasoning, nondeterministic outputs, and complex interaction dynamics, and reliable benchmarks are needed to guide attribution methods.

Problem

Existing attribution benchmarks use only partially observable traces that capture agent outputs but omit the inputs and context developers actually rely on when debugging, obscuring true failure causes.

Motivation / Analysis

Attribution should be studied under full execution observability, matching real developer scenarios where complete traces are available, so that methods can be evaluated fairly.

Key Insight

Missing inputs, not just outputs, hide many failure causes; giving attribution methods full traces plus replayable environments dramatically improves their ability to localize the responsible agent and decisive step.

Challenges

Capturing complete, reproducible execution traces for nondeterministic MAS and constructing environments that can be replayed to test attribution techniques systematically.

Method

TraceElephant supplies full execution traces and reproducible environments, then systematically evaluates prompting-based and agent-based attribution techniques across configurations.

Experimental Setup

Attribution techniques are evaluated on TraceElephant comparing full-observability traces against a partial-observation counterpart.

Results

Full traces boost attribution accuracy by up to 76% versus the partial-observation setting, showing that missing inputs obscure many failure causes; the benchmark is offered as a foundation for follow-up failure-attribution research. Accepted by ACL 2026.

Limitations

The benchmark measures attribution quality rather than preventing failures; requiring full traces and reproducible environments may be costly and not always available in production MAS.

Relation to the Field

Provides the attribution counterpart to MAST's failure taxonomy and AgentCollabBench's failure detection; closely related to TELBench/DRIFT span-level localization for deep-research agents.
