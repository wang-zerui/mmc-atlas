# Self-Consistent Mixture of Agents / Trace-Level Synthesis ## 一句话 - 主张聚合的单元应当是推理 TRACE（轨迹）而非最终答案：一个读取完整轨迹的聚合器即使在全体一致同意的情况下也能恢复出正确解（即“聚合悖论”），并且单模型扰动轨迹优于异质模型池。 ## 问题 - 压缩为投票会丢弃中间的推理信息，且当所有 agent 都收敛到同一个错误答案时无法恢复出正确解。 ## 洞见 - “聚合悖论”：一个读取完整轨迹的聚合器即使在所有 agent 一致同意时也能恢复出正确答案；是误差相关性、而非单纯的多样性，限制了 majority voting 的上限，因此轨迹级别的互补性才是真正的杠杆。 ## 方法 - Self-Consistent Mixture of Agents 通过对单个模型施加语义保持的输入扰动来制造轨迹多样性，始终在完整轨迹上进行合成（从不以共识为门控），并使用带有可证明非退化保证的“anchored refinement”（锚定精炼）来保护多数答案。 ## 设置 - 在四个领域（结构化推理、博士级科学、竞赛数学、竞技编程）上进行评测，将单个扰动模型与异质模型池进行对比。 ## 结果 - 在全部四个领域中，带扰动引发轨迹变化的单个模型都优于异质模型池，且额外增加的扰动多样性并不会抬高投票上限，因为误差相关性完全相同（已与摘要核实；摘要中无数值基准）。 ## 挑战 - 防止一个进行合成的聚合器把本已正确的多数答案改坏，以及在无需异质模型的情况下生成有用的轨迹多样性。 ## 局限 - 摘要未报告任何数值基准分数，且 anchored-refinement 的保证建立在所陈述的假设之上；这是一篇 2026 年的预印本。

背景

Mixture-of-Agents 通常将多个 agent 的推理压缩为对最终答案的 majority vote 或分层合成。

问题

压缩为投票会丢弃中间的推理信息，且当所有 agent 都收敛到同一个错误答案时无法恢复出正确解。

动机 / 分析

如果有益的修正能够可靠地多于有害的修正，那么一个读取完整推理轨迹的聚合器就能合成出被投票所丢弃的正确性。

关键洞见

“聚合悖论”：一个读取完整轨迹的聚合器即使在所有 agent 一致同意时也能恢复出正确答案；是误差相关性、而非单纯的多样性，限制了 majority voting 的上限，因此轨迹级别的互补性才是真正的杠杆。

挑战

防止一个进行合成的聚合器把本已正确的多数答案改坏，以及在无需异质模型的情况下生成有用的轨迹多样性。

方法

Self-Consistent Mixture of Agents 通过对单个模型施加语义保持的输入扰动来制造轨迹多样性，始终在完整轨迹上进行合成（从不以共识为门控），并使用带有可证明非退化保证的“anchored refinement”（锚定精炼）来保护多数答案。

实验设置

在四个领域（结构化推理、博士级科学、竞赛数学、竞技编程）上进行评测，将单个扰动模型与异质模型池进行对比。

结果

在全部四个领域中，带扰动引发轨迹变化的单个模型都优于异质模型池，且额外增加的扰动多样性并不会抬高投票上限，因为误差相关性完全相同（已与摘要核实；摘要中无数值基准）。

局限

摘要未报告任何数值基准分数，且 anchored-refinement 的保证建立在所陈述的假设之上；这是一篇 2026 年的预印本。
