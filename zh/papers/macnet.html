# MacNet / Scaling LLM Multi-Agent Collaboration ## 一句话 - MacNet 将 LLM agent 组织为有向无环图（DAG），可扩展到 1000+ 个协作 agent，并识别出一条呈 logistic 质量增长的“协作 scaling law”，其中 small-world/不规则拓扑表现最佳。 ## 问题 - 此前的多 agent 工作只使用少数几个 agent，也没有刻画协作如何随 agent 数量或拓扑而扩展。 ## 洞见 - 将 agent 排布在 DAG 上可协调跨拓扑的推理；整体质量遵循一条 logistic 的“协作 scaling law”，且协作涌现比神经网络涌现出现得更早，不规则拓扑优于规则拓扑。 ## 方法 - MacNet（多 agent 协作网络）将 agent 置于有向无环图上，并沿边路由交互式的反思/精炼，支持超过一千个 agent 之间的协作。 ## 设置 - 扩展 agent 数量并改变网络拓扑的实验，并拟合出一条协作 scaling law。 ## 结果 - 已核实的贡献：支持 >1000 个 agent；不规则/small-world 拓扑优于规则拓扑；性能遵循一条带早期涌现的 logistic 协作 scaling law。各 benchmark 的具体量级见论文正文，而非摘要。 ## 挑战 - 在不发生规模爆炸的前提下协调众多 agent 的推理，并找出哪些拓扑最有利于交互式的反思与精炼。 ## 局限 - 收益在 logistic 律下会饱和（大 N 时边际递减）；收益取决于拓扑选择，且摘要未给出单一标志性准确率数字。

背景

神经网络 scaling law 表明性能随模型规模提升；而增加协作 agent 数量是否也有类似效果，此前尚未可知。

问题

此前的多 agent 工作只使用少数几个 agent，也没有刻画协作如何随 agent 数量或拓扑而扩展。

动机 / 分析

理解 agent 数量的扩展规律与拓扑影响，将有助于指导如何高效地构建大规模协作系统。

关键洞见

将 agent 排布在 DAG 上可协调跨拓扑的推理；整体质量遵循一条 logistic 的“协作 scaling law”，且协作涌现比神经网络涌现出现得更早，不规则拓扑优于规则拓扑。

挑战

在不发生规模爆炸的前提下协调众多 agent 的推理，并找出哪些拓扑最有利于交互式的反思与精炼。

方法

MacNet（多 agent 协作网络）将 agent 置于有向无环图上，并沿边路由交互式的反思/精炼，支持超过一千个 agent 之间的协作。

实验设置

扩展 agent 数量并改变网络拓扑的实验，并拟合出一条协作 scaling law。

结果

已核实的贡献：支持 >1000 个 agent；不规则/small-world 拓扑优于规则拓扑；性能遵循一条带早期涌现的 logistic 协作 scaling law。各 benchmark 的具体量级见论文正文，而非摘要。

局限

收益在 logistic 律下会饱和（大 N 时边际递减）；收益取决于拓扑选择，且摘要未给出单一标志性准确率数字。
