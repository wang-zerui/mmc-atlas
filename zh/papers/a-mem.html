A-MEM: Agentic Memory for LLM Agents (Xu et al. 2025)

M099NeurIPS 2025已核实 原文 ↗
A-MEM 是一种受 Zettelkasten 启发的 agentic memory，其中每条 memory 都是一个原子笔记，带有 LLM 生成的关键词、标签和上下文描述，动态链接到相关笔记，并允许随新 memory 到来而演进——超越了静态的 RAG 检索。

背景

LLM agent 依赖 memory 系统在多次交互间持久保存和回忆信息，但大多数系统只提供基础的存储和检索。

问题

现有 memory 系统缺乏精细的组织：它们存储和取回条目，却不构建或维护一个结构化、相互关联的知识网络。

动机 / 分析

Zettelkasten 笔记法展示了原子化、相互链接的笔记如何形成一个不断演进的知识网络，这为 agent memory 提供了更丰富的组织原则。

关键洞见

将 Zettelkasten 结构与 agent 驱动的决策相结合，可让 memory 自我组织：新笔记生成自己的属性、链接到语义相关的过往笔记，并能触发对现有 memory 属性的更新。

挑战

决定链接哪些过往 memory、自动生成有用的结构化属性，以及在不破坏已有知识的前提下将更新在网络中传播。

方法

对每条新 memory，系统生成一个结构化笔记（上下文描述、关键词、标签），在存在有意义相似性之处将其链接到现有 memory，并让此次新增触发 memory 演进，更新相关历史笔记的表示和属性。

实验设置

在六个基础模型上进行了实证评估，并与现有的 state-of-the-art memory baseline 进行对比。

结果

论文报告在六个基础模型上相比现有 SOTA memory baseline 有显著提升；摘要中未给出具体的逐项 benchmark 指标值。已验证：动态索引/链接加上 memory 演进、六模型评估、NeurIPS 2025。

局限

摘要没有报告提升的确切幅度，也没有报告 LLM 驱动的笔记生成与演进的成本/延迟；收益可能随基础模型的结构化能力而变化。

与领域的关系

作为以结构和演进为中心的替代方案，与以 context 分页为核心的 memory（如 MemGPT）形成对照；两者都是单 agent memory 系统，而 Mesh Memory Protocol 处理的是跨 agent 的 memory 共享。
