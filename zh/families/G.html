G. 通信与记忆协议

9 篇论文
规定实例之间如何传消息、共享状态、暴露工具的标准。

基础方法。 MCP (M020) 标准化了 agent↔tool；A2A (M095) 标准化 agent↔agent；一篇综述
(M096) 梳理了 MCP/ACP/A2A/ANP；记忆基底 MemGPT (M098) 与 A-MEM
(M099)。2026 SOTA。 Beyond-Message-Passing (M097) 形式化了一个 3 层
（comm/syntax/semantics）分类法，并表明当前协议对*语义*
对齐支持不足（澄清/修复被以技术债的形式推给了 prompt）；
Web-of-Agents（M212，ICML'26 立场文）主张需要最小化的共享标准
以避免碎片化；ProtocolBench (M211) 隔离出协议效应（完成
时间差异最高达 36.5%）——不过其 ICLR'26 状态仍在审稿中，尚未
确认；Mesh Memory (M213) 提出跨会话的认知共享。协议
在语法上正走向成熟；尚未填补的缺口是语义。

本族论文

Beyond Message Passing: A Semantic View of Agent Communication Protocols (Yuan et al. 2026)
本文将 agent 通信 protocol 组织为受 Shannon-Weaver 和 Grice 语用学启发的三层分类体系（communication、syntactic、semantic），并指出当前 protocol 在传输/语法层已趋成熟，但对澄清与修复等语义对齐的支持严重不足。
M097arXiv 2026 (preprint)
ProtocolBench / ProtocolRouter (Du, Zhu et al. 2025/2026)
ProtocolBench 通过在保持 LLM、prompt 和硬件不变的同时改变 protocol（A2A/ACP/ANP/Agora）来隔离 agent 通信 protocol 的影响，而 ProtocolRouter 是一个可学习的按模块路由器，从需求和运行时信号中选择 protocol。
M211ICML 2026 (accepted per arXiv v3, Jun 2026)
Position: Collaborative Agentic AI Needs Interoperability Across Ecosystems (Sharma et al. 2026)
这篇 ICML 2026 立场论文主张，协作式 agentic AI 需要最小化的共享标准以避免碎片化为互不兼容的生态系统，并提出以四个组件构建的“Web of Agents”作为最小化的架构基础。
M212ICML 2026 (position paper)
Mesh Memory Protocol (Xu 2026)
Mesh Memory Protocol (MMP) 是一个被提出的语义基础设施层，让 LLM agent 团队能够跨会话共享、评估并组合彼此的认知状态，由四个可组合原语（CAT7、SVAF、inter-agent lineage、remix）构成。
M213arXiv 2026 (v0.2.3 specification, CC BY 4.0)
A2A (Agent2Agent) Protocol (Google 2025)
A2A 是一个开放的点对点 protocol，让基于不同 framework、由不同厂商构建的 AI agent 能够相互发现、委派任务并共享结果，而无需暴露各自的内部 memory、工具或逻辑。
M095Open standard (Google, Apr 2025); donated to the Linux Foundation Jun 2025
Survey of Agent Interoperability Protocols (Ehtesham et al. 2025)
一篇综述，从交互、发现、通信和安全四个维度对比四种新兴的 agent 互操作性 protocol——MCP、ACP、A2A 和 ANP——并提出一条分阶段的采用路线图，用于构建可互操作的 LLM-agent 生态系统。
M096arXiv 2025 (preprint)
A-MEM: Agentic Memory for LLM Agents (Xu et al. 2025)
A-MEM 是一种受 Zettelkasten 启发的 agentic memory，其中每条 memory 都是一个原子笔记，带有 LLM 生成的关键词、标签和上下文描述，动态链接到相关笔记，并允许随新 memory 到来而演进——超越了静态的 RAG 检索。
M099NeurIPS 2025
Model Context Protocol (Anthropic 2024)
MCP 是一个开放的 JSON-RPC 标准，被比喻为“AI 的 USB-C 接口”，通过统一的 client-server 接口将 AI 应用连接到外部数据源、工具和工作流。
M020Open standard (Anthropic, Nov 2024); donated to the Agentic AI Foundation (AAIF) Dec 2025
MemGPT (Packer et al. 2023)
MemGPT 将操作系统的内存层级思想应用于 LLM，让模型通过 function call 自我管理分层 memory——固定的 main context（类似 RAM）加上外部存储（类似 disk）——从而表现得仿佛拥有远大于实际的 context window。
M098arXiv 2023 (later COLM 2024); now the Letta framework
